## Design Implementation
The current portfolio implements a Contra game theme with:
- Retro 8-bit/16-bit pixel art aesthetics
- Military jungle color scheme (greens, oranges, browns)
- Classic arcade game UI elements
- Neon green (#00ff00) terminal/HUD style
- Pixelated fonts and borders
- Game-inspired interactions and animations

## Theme Elements
- **Colors:** Military green (#2d5016), Neon green (#00ff00), Orange-red (#ff6b35)
- **Fonts:** 'Press Start 2P' (pixel font), 'VT323' (retro monospace)
- **Style:** Retro terminal cards, pixel borders, scanline effects
- **Interactive:** Particle explosions, power-up animations, Konami code easter egg
